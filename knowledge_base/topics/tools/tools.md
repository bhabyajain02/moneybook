# Developer Tools & Libraries

**Tags:** tools
**First learned:** 2026-03-28
**Last updated:** 2026-03-28

---

### Web Frameworks
*   **FastAPI**: A modern, fast (high-performance) web framework for building APIs with Python, offering automatic interactive API documentation.
### Databases
*   **SQLite**: A self-contained, serverless, zero-configuration, transactional SQL database engine, ideal for embedded use or local development.
### Messaging Platforms
*   **Twilio Sandbox for WhatsApp**: Provides a testing environment for developing WhatsApp bots and integrations without needing to go through the full WhatsApp Business API approval process.
### AI/ML
*   **Anthropic Claude API**: Offers advanced large language models, including vision-enabled models (Sonnet) for image understanding and text models (Haiku) for natural language processing.
### Local Tunneling
*   **Cloudflared Tunnel**: Securely exposes local development servers to the internet without configuring firewall rules, useful for testing webhooks with external services. Be aware of dynamic URLs on restart.
### Task Scheduling
*   **APScheduler**: A Python library that lets you schedule your Python functions to be executed at a specific time or repeatedly.
